什么是委托类型

发表于 2020-02-20 分类于 C#

委托（delegate） 是一种类型安全的函数指针。在C#中，delegate 是一种类型，包装了一个或多个函数指针及绑定的类实例。委托用来实现函数回调与事件接收（event listener）。Delegate对象可以作为参数传递给其他函数，以引用（referenced）封装在delegate对象中的函数，而无需在编译时刻就绑定被调用函数。

一旦为委托分配了函数方法，委托将与该函数方法具有完全相同的行为。 与委托的类型特征（由返回类型和参数组成）匹配的任何方法都可以分配给该委托。

什么是面向对象编程

面向对象编程（Object Oriented Programming）,简称 OOP，是一种程序设计思想。OOP 把 对象 作为程序的基本单元，一个对象包含了 数据 和操作数据的 方法。

面向过程的程序设计把程序视为一系列的命令集合，即一组函数的顺序执行。为了简化程序设计，面向过程把函数继续切分为子函数，即把大块函数通过切割成小块函数来降低系统的复杂度。

面向对象的程序设计把程序视为一组对象的集合，而每个对象都可以接收其他对象发过来的消息，并处理这些信息，计算机程序的执行就是一系列消息在各个对象之间传递。

面向对象的设计思想是抽象出事物的 类（Class），根据 类 创建出具体对象的 实例（Instance）。

面向对象编程有三大特点：封装、继承、多态

封装

类 是事物的抽象，具有事物的属性、行为。

实现：设置属性的访问权限，提供属性的访问方法。

继承

通过继承，拥有基类的所有非私有属性和所有非私有方法。

多态

多态 指允许不同类的对象对同一消息做出不同响应。

C# 中的值类型和引用类型

发表于 2020-02-15 更新于 2020-04-20 分类于 C#

两者对比总结

值类型 引用类型
变量存储内容 数据 数据的引用
数据存储位置 栈 堆

值类型和引用类型是C#主要的两个类别。
它们的不同在于：
一个值类型的变量直接在栈中存储数据本身。而引用类型的数据存放在堆中，引用类型的变量中存储的是一个指向具体数据的引用。
所以，当把值类型 变量A 赋值给值类型 变量B 后，改变 变量B 的值不会影响 变量A 的值。而当把值类型 变量C 赋值给值类型 变量D 后，改变 变量D 的值会影响 变量C 的值，因为 变量C 和 变量D 指向同一个内存空间。
